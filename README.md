# кинокультура — Telegram-бот

Бот для киноклуба [кинокультура](https://t.me/oldtownuu) в Улан-Удэ. Помогает зрителям записываться на показы и получать афишу, а организаторам — управлять списками гостей и делать рассылки.

---

## Возможности

### Для зрителей
| Кнопка | Что делает |
|---|---|
| 📰 афиша месяца | Показывает актуальную афишу с ссылкой на пост в канале |
| 🎟 мои встречи | Список показов, на которые записан пользователь |
| 🛎️ наш адрес | Адрес площадки (пост из канала с картой или видео) |
| 🎞 о кинокультуре | Описание проекта |

Запись на показ происходит через deep link — кнопка «я приду на встречу» в канале ведёт прямо в бот.

### Для администратора
| Кнопка | Что делает |
|---|---|
| ➕ Создать анонс | Пошаговый мастер: создаёт сеанс, добавляет кнопку к посту в канале и делает рассылку всем пользователям |
| 📌 Обновить афишу | Обновляет афишу (пересылка фото из канала) |
| 📍 Обновить адрес | Обновляет адрес (пересылка поста из канала) |
| 👥 Списки гостей | Список записавшихся по каждому сеансу |
| 📢 Рассылка (Текст) | Текстовая рассылка всем пользователям |
| 🗑 Удалить сеанс | Удаляет сеанс из базы |

Для отмены любого действия — `/cancel`.

## Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/osborne-cox/kino-bot.git
cd kino-bot
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Настроить окружение

```bash
cp .env.example .env
```

Открыть `.env` и заполнить:

```env
BOT_TOKEN=токен_от_BotFather
ADMIN_ID=ваш_telegram_id
BOT_USERNAME=username_бота_без_собаки
```

Получить `BOT_TOKEN` → [@BotFather](https://t.me/BotFather)  
Узнать свой `ADMIN_ID` → [@userinfobot](https://t.me/userinfobot)

### 4. Запустить

```bash
python bot.py
```

## Структура файлов

```
kino-bot/
├── bot.py              # основной код бота
├── .env                # токен и настройки (не коммитится)
├── .env.example        # шаблон для .env
├── requirements.txt
├── sessions.json       # база сеансов и записей (создаётся автоматически)
├── all_users.json      # список всех пользователей для рассылок
├── afisha.json         # текущая афиша
└── address.json        # адрес площадки
```

JSON-файлы создаются автоматически при первом запуске и **не должны попадать в git** (прописаны в `.gitignore`).

## Как создать новый показ

1. Опубликовать пост с анонсом (с картинкой) в Telegram-канале
2. В боте нажать **➕ Создать анонс**
3. Следовать шагам мастера: ввести ID, название, переслать пост из канала
4. Бот автоматически добавит к посту кнопку «я приду на встречу» и разошлёт анонс всем пользователям
